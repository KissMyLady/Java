
## ASCII

```Java
package to.today.my02;

public class ascii {
    public static void main(String[] args){
        int a = 10;
        char b = 'A';
        System.out.println(a +b);
    }
}
```
输出: `75`

将所有的英文字母，数字，符号都和十进制进行了对应，
因此产生了世界上第一张编码表 
ASCII(American Standard Code for Information Interchange)美国标准信息交换码
